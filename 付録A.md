### 三証明主義

|用語|読み方|意味|
|---|---|---|
|三証明主義|さんしょうめいしゅぎ|本世界の司法における三つの証明方法（証拠主義、神託証明主義、魔法的真実証明主義）の総称|
|証拠主義|しょうこしゅぎ|三証明主義の一つ。物理的証拠・論理的推論に基づいて事実を証明する方法。正式名称は「物的証拠証明主義」|
|物的証拠証明主義|ぶってきしょうこしょうめいしゅぎ|証拠主義の正式名称|
|神託証明主義|しんたくしょうめいしゅぎ|三証明主義の一つ。神からの啓示に基づいて事実を証明する方法。正式名称は「神聖啓示証明主義」|
|神聖啓示証明主義|しんせいけいじしょうめいしゅぎ|神託証明主義の正式名称|
|魔法的真実証明主義|まほうてきしんじつしょうめいしゅぎ|三証明主義の一つ。魔法による現場再現に基づいて事実を証明する方法。正式名称は「魔術的再現証明主義」|
|魔術的再現証明主義|まじゅつてきさいげんしょうめいしゅぎ|魔法的真実証明主義の正式名称|
|第一判定|だいいちはんてい|三証明主義における優先順位で、物理的時間・証拠主義による判定|
|第二判定|だいにはんてい|三証明主義における優先順位で、神託証明主義・魔法的真実証明主義による補助的判定|
|副次的証明|ふくじてきしょうめい|三証明主義において、証拠主義を補強する役割を持つ神託証明主義と魔法的真実証明主義|
|確からしさ|たしからしさ|本世界の裁判における判決基準。絶対的真実ではなく、証拠や証明から導かれる蓋然性に基づいて判断する|
|確からしさ基準|たしからしさきじゅん|裁判において絶対的真実の証明を要求せず、確からしさに基づいて判決を下すという原則|
|物的証拠|ぶってきしょうこ|凶器、遺留品、現場の痕跡など物理的に存在する証拠|
|状況証拠|じょうきょうしょうこ|直接的ではないが事実を推認させる証拠|
|目撃証言|もくげきしょうげん|事件を直接見た者の証言。証拠主義における証拠の一種|
|自白|じはく|被告人が自らの犯行を認めること。証拠の一種だが、自白のみでは有罪認定に不十分とされる|
|現場再現|げんばさいげん|魔法的真実証明において、犯罪現場と同等の状況を再現すること。証明の発動条件となる|
|物理的時間|ぶつりてきじかん|物証・目撃証言・記録による客観的な時間。同時多発性判定の基礎となる|
|神聖な時間|しんせいなじかん|神託証明における時間概念。物理的時間とは異なる「神の時間」|
|精神的時間|せいしんてきじかん|魔法的真実証明における時間概念。動機形成時期など精神的な時間軸|
|論理性|ろんりせい|因果関係や推論の筋道。魔法的真実証明主義はこれを欠くという弱点がある|

### 罪と刑罰

|用語|読み方|意味|
|---|---|---|
|個人罪|こじんざい|単独で犯した罪、または同時多発性が認められない罪。通常の刑罰（×1.0）が科される|
|集団罪|しゅうだんざい|2名以上が同時多発的に同じ罪を犯した場合に成立する罪。刑罰は通常の1.75倍となる|
|同時多発性|どうじたはつせい|集団罪の成立要件の一つ。複数の犯罪が同時期に発生していること|
|個人責任主義|こじんせきにんしゅぎ|罪は個人に帰属し、種族全体や血縁者に連座しないという原則|
|種族不問|しゅぞくふもん|犯罪者・被害者の種族に関わらず、同一の法体系で裁かれるという原則|
|連座|れんざ|本人以外の者（血縁者・種族全体など）が罪の責任を負わされること。本世界では個人責任主義によりこれを否定する|
|1.75倍|いってんななごばい|集団罪に適用される刑罰係数。個人責任分（×1.0）に社会的影響加算分（+0.75）を加えた値|
|2名|にめい|集団罪が成立する最低人数|

|裁判官|さいばんかん|法廷を主宰し、判決を下す司法従事者。異種族混合法廷では複数種族で構成される|
|検察官|けんさつかん|犯罪を捜査し、起訴する司法従事者|
|弁護人|べんごにん|被告の権利を守り、弁護する司法従事者|

|経験ルート|けいけんるーと|全法織の長への出馬資格を得る方法の一つ。10年以上の司法従事者としての実務経験が必要|

|フィードバック|ふぃーどばっく|特殊司法官試験において、各地域の司法従事者が受験者を評価して全法織に提出する報告|

### 裁判手続き

| 用語   | 読み方       | 意味                                                        |
| ---- | --------- | --------------------------------------------------------- |
| 起訴   | きそ        | 検察官が被疑者を裁判にかけること。起訴により被疑者は被告人となる                          |
| 不起訴  | ふきそ       | 検察官が被疑者を裁判にかけないと判断すること。証拠不十分などの理由による                      |
| 公判   | こうはん      | 裁判所で行われる審理手続き                                             |
| 公開原則 | こうかいげんそく  | 裁判は原則として公開で行われるという原則                                      |
| 人定質問 | じんていしつもん  | 公判の冒頭で裁判官が被告人の氏名・住所等を確認し、本人であることを特定する手続き                  |
| 黙秘権  | もくひけん     | 被告人が自己に不利益な供述を強要されない権利。公判冒頭で裁判官から告知される                    |
| 罪状認否 | ざいじょうにんぴ  | 公判冒頭で被告人が起訴状に対する認否（認めるか否か）を述べる手続き                         |
| 冒頭陳述 | ぼうとうちんじゅつ | 公判の冒頭で検察側・弁護側がそれぞれの主張の概要を述べること                            |
| 論告   | ろんこく      | 公判において検察官が証拠に基づく有罪の主張と求刑を行うこと                             |
| 弁論   | べんろん      | 公判において弁護人が無罪または減刑を主張すること                                  |
| 求刑   | きゅうけい     | 論告において検察官が裁判所に対して具体的な刑罰を求めること                             |
| 上訴   | じょうそ      | 判決に不服がある場合に上級審に審理を求めること                                   |
| 棄却   | ききゃく      | 訴訟要件の欠如等により審理に入らず終了させること。日本法における「却下」に相当する概念を、本世界では「棄却」と呼ぶ |
| 却下   | きゃっか      | 日本法における用語。訴訟要件を欠く場合に審理に入らず訴えを退けること。本世界では同概念を「棄却」と呼称する     |
| 再審制度 | さいしんせいど   | 新たな証拠が発見された場合に、確定した判決を再度審理する制度                            |
