## FAQ：よくある質問と回答

### 三証明主義に関するFAQ

|No.|質問|回答|
|---|---|---|
|1|三つの証明が矛盾したらどうなる？|証拠主義が基礎、神託・魔法的真実が副次という優先順位に従う。ただし副次証明が極めて強力な場合は考慮される。最終的には裁判官の合議で「確からしさ」を判定する|
|2|被害者が複数いて信仰がバラバラな場合は？|各被害者の信奉する神それぞれに交信を行う。複数の神託が得られ、それらを総合的に判断する|
|3|被害者が死亡していて信仰が不明な場合は？|遺族や関係者への聞き取り、生前の記録などから信仰を特定する。特定できない場合は神託証明を使用できない|
|4|魔法的真実証明の「現場再現」はどこまで厳密？|場所、時間帯、天候、周囲の状況など可能な限り再現する必要がある。完全一致は求められないが、著しく異なる場合は証明力が低下する|
|5|神託で「神が沈黙した」場合はどうなる？|神託証明としては「証明不能」となる。他の証明方法で補う必要がある|
|6|神官が買収されていた場合の対応は？|複数名の同時確認制度により防止を図っている。発覚した場合、その神官は資格剥奪となり、神託証明は無効化される。ただし全員を買収された場合の完全な防止策はない|
|7|魔法的真実証明を妨害・改竄する方法はある？|理論上は存在しうる。記憶改竄魔法や現場への魔術的干渉など。そのため魔法証明士の資格審査は厳格であり、妨害が疑われる場合は証明力が低下する|

### 集団罪に関するFAQ

| No. | 質問                        | 回答                                                                                                                          |
| --- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| 8   | 「同時多発的」の時間幅はどのくらい？        | 明確な数値基準はなく、事件の性質や状況により判断される。これ自体が法廷での争点になりうる                                                                                |
| 9   | 2名のうち1名が無罪になったら集団罪は解除される？ | 同一裁判内または並行裁判において、2名のうち1名が無罪となった場合は解除される。集団罪は「2名以上が有罪」であることが前提のため、1名になった時点で残った者は個人罪として再計算される。ただし、既に判決が確定している場合は再審手続きを経る必要がある |
| 10  | 3名以上が集団罪の場合、1名が無罪になったら？   | 残り2名以上が有罪であれば集団罪は維持される。1.75倍の係数もそのまま適用                                                                                      |
| 11  | 集団罪の1.75倍はなぜこの数字？         | 個人責任分（×1.0）を維持しつつ、社会的影響の増大分（+0.75）を加算するという法理論に基づく。学術的議論を経て定められた係数                                                           |
| 12  | 同じ罪を犯したが動機が全く違う場合も集団罪？    | 集団罪の成立に意図や動機の共有は不要。罪名の同一性、複数性、同時多発性の三要件のみで判定される                                                                             |
| 13  | 集団罪の適用を巡って判例の蓄積はある？       | ある。特に「同時多発性」の解釈については多くの判例があり、弁護側・検察側双方が過去の判例を引用して争う                                                                         |

### 全法織に関するFAQ

|No.|質問|回答|
|---|---|---|
|14|全法織の長が暴走したら止める仕組みはある？|評議会による不信任決議、5年ごとの選挙による審判がある。ただし任期中の罷免は極めて困難|
|15|派閥に属さない司法従事者はいる？|いる。ただし出世や選挙での支援を考えると、どこかの派閥と協力関係を持つ者が多い。完全な中立は政治的に不利になりやすい|
|16|派閥間で連立を組むことはある？|ある。特に選挙時には、単独で過半数を取れない派閥同士が連立を組むことがある。証拠至上派と魔法信頼派の協調などが典型例|
|17|評議会の構成はどうなってる？|各種族の司法機関から選出された代表者で構成される。種族間のバランスが考慮されるが、その配分自体が政治的議論の対象となる|
|18|全法織と各種族の司法機関の上下関係は？|全法織は「統合・調整」機関であり、各種族の司法機関を支配する上位機関ではない。種族内事件は各種族の管轄、種族間事件は全法織の管轄という分担|
|19|全法織の予算や運営費はどこから出る？|各種族の司法機関からの分担金で運営される。分担比率は評議会で決定され、これも政治的な争点になりうる|

### 特殊司法官試験に関するFAQ

|No.|質問|回答|
|---|---|---|
|20|試験中に冤罪を出したら即不合格？|即不合格とは明記されていないが、「一発アウト級」の評価項目であり、事実上の不合格となる可能性が極めて高い|
|21|試験中の判決は後から取り消せる？|通常の判決と同様に上訴・再審の対象となる。試験だからといって特別扱いはされない|
|22|試験の合格率はどのくらい？|公式な数値は非公開だが、極めて低いとされる。1年間の過酷な試験を完走できる者自体が少ない|
|23|不合格になったら再受験できる？|再受験は可能。ただし前回の評価記録は残るため、同じ失敗を繰り返せば合格は困難になる|
|24|試験中に担当する事件は選べる？|選べない。全法織が派遣先と担当事件を決定する。受験者の得意・不得意に関係なく、多様な事件を経験させる方針|
|25|試験中に派閥から支援を受けることは？|禁止されていないが、露骨な支援は他の評価者からの心証を悪くする。また、特定派閥との癒着と見なされるリスクがある|

### 種族に関するFAQ

| No. | 質問                                | 回答                                                                                                                             |
| --- | --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| 26  | 人間以外の種族が全法織の長になったことはある？           | 設定上は可能。選挙で選ばれれば種族に関係なく就任できる。歴史上の就任実績は創作者が自由に設定してよい                                                                             |
| 27  | 種族間で法の解釈が対立したらどうなる？               | 異種族混合法廷での合議により調整される。それでも対立が解消しない場合は、全法織の評議会で政策的に議論される                                                                          |
| 28  | 異種族間の結婚で生まれた子の裁判管轄は？              | 居住地の司法機関が第一管轄となる。ただし種族的アイデンティティに関わる事件では、関係する種族の司法機関との協議が行われることもある                                                              |
| 29  | 種族ごとに刑罰の重さは変わる？                   | 原則として変わらない。種族不問の原則に基づき、同一の罪には同一の刑罰が科される。ただし種族の生態に基づく刑の執行方法の調整はありうる。また、種族固有の刑罰（エルフの森との断絶、ドワーフの鍛造権剥奪など）が存在し、該当種族の被告にのみ適用されることがある |
| 30  | 寿命が極端に違う種族間で「同時多発的」はどう判断する？       | 物理的時間を基礎とするため、種族の寿命は考慮されない。エルフの「一瞬」と人間の「一瞬」が異なっても、客観的な時間軸で判定される                                                                |
| 31  | 種族固有の犯罪（エルフの森の法違反など）に他種族が問われることは？ | ありうる。ただしその場合、当該種族の法に精通した裁判官が合議に参加することが求められる                                                                                    |

### 司法取引に関するFAQ

|No.|質問|回答|
|---|---|---|
|32|司法取引を拒否したら不利になる？|法的には不利にならない建前だが、実務上は「争う姿勢」と見なされ、心証に影響する可能性は否定できない|
|33|司法取引の内容は公開される？|原則として判決時に公開される。ただし内容の詳細は「合意の存在」のみが記録され、交渉過程は非公開|
|34|司法取引を破ったらどうなる？|被告側が破った場合、取引は無効となり元の起訴内容で裁判が再開される。検察側が破った場合、被告は取引内容の履行を求めて異議申立てができる|
|35|司法取引は弁護人なしでも成立する？|法的には可能だが、推奨されない。弁護人なしの取引は「強要の可能性」を理由に、後から無効を主張されるリスクがある|
|36|冤罪を生んだ司法取引が発覚した場合の責任は？|検察官が不当な圧力をかけていた場合、懲戒処分の対象となりうる。ただし「自発的な自白」と認定された場合、責任追及は困難|

### 創作向けFAQ

| No. | 質問                     | 回答                                                                                                                  |
| --- | ---------------------- | ------------------------------------------------------------------------------------------------------------------- |
| 37  | 主人公を弁護人にする場合のおすすめ展開は？  | 集団罪の適用阻止、三証明の矛盾を突く、無神論者被害者の事件で証拠主義一本勝負、派閥間の政治に巻き込まれる、などが定番                                                          |
| 38  | どの派閥が「悪役」にしやすい？        | どの派閥も悪役にできる。神託重視派の「異端審問的」側面、証拠至上派の「冷酷な合理主義」、三柱均衡派の「保身的現状維持」など、切り口次第                                                 |
| 39  | この世界観を改変して使ってもいい？      | 自由に改変・再配布してよい（Creative Commons）。種族の追加、新派閥の創設、第四の証明方法の導入など、創作者の世界観に合わせてカスタマイズ推奨                                      |
| 40  | 検察官を主人公にする場合の展開は？      | 正義のために集団罪を立証しようとするが、途中で冤罪の可能性に気づく。派閥の圧力と自身の信念の間で葛藤する、などが効果的                                                         |
| 41  | 被告人を主人公にする場合の展開は？      | 無実の罪で起訴され、三証明全てが不利な状況から逆転を目指す。または実際に罪を犯したが、その背景にある事情を明かしていく構成                                                       |
| 42  | 特殊司法官試験を題材にした物語のポイントは？ | 1年間の試験期間をロードムービー的に描く。各地での事件、派閥のバイアス、冤罪未遂の危機、成長と挫折を織り交ぜる。合格がゴールではなく「どんな司法従事者になるか」をテーマにすると深みが出る                       |
| 43  | 刑罰の種類やランクを改変して使ってもいい？  | 自由に改変してよい。ランクの増減、種族固有刑罰の追加・削除、係数の変更、極刑の廃止など、創作者の世界観に合わせてカスタマイズ推奨。執行方法の調整例も一例に過ぎないため、種族の文化設定に応じて独自の執行方法を設計するとより深みが出る |
